============
Introduction
============


This Documentation was written for version 2.0.x of the extension.


What does it do?
----------------


Planing and display tool for session plans:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

In the backend a module assists the planing process by adding sessions
in a grid and helps keeping an overview of which slots are booked and
which are free.

In frontend the session plan is rendered in a table style.
